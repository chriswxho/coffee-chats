# coffee-chats
Coffee chat matching app

## Instructions
1) Setup by running 
```
sh setup.sh
```
2) Run the coffee-matching app by running
```
sh main.sh <participants file> <results file>
```

For example:
```
sh main.sh participants/nov24.csv nov24_test.csv
```

You might run into permission errors.
```
chmod +x main.sh
chmod +x setup.sh
```
